Kali Linux源使用帮助

Kali Linux 源使用帮助

地址

https://mirrors.zzu.edu.cn/kali/

说明

Kali Linux 软件源

支持的系统架构

amd64, armel, armhf, i386

使用说明

编辑 /etc/apt/sources.list 文件, 在文件最前面添加以下条目：

deb https://mirrors.zzu.edu.cn/kali kali-rolling main non-free contrib
	deb-src https://mirrors.zzu.edu.cn/kali kali-rolling main non-free contrib

更改完 sources.list 文件后请运行 sudo apt-get update 更新索引以生效。
